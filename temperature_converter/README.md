# Temperature Converter

## Description

A C++ console application that converts temperatures between Celsius and Fahrenheit.

---

## Features

* Converts Celsius to Fahrenheit
* Converts Fahrenheit to Celsius
* Uses a menu for user selection
* Allows repeated conversions

---

## Concepts Practed

* Conditional statements (`if`, `else if`)
* Arithmetic formulas
* User input handling
* Loops (`do-while`)

---

## How to Run

Compile and run the program using a C++ compiler.
