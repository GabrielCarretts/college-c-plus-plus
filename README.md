# College C++ Projects

## Description

This repository contains small C++ programs developed during my first semester of college.

Each project focuses on practicing fundamental programming concepts such as input/output, arithmetic operations, and control structures.

---

## Projects

* **Simple Calculator** – basic arithmetic operations
* **Temperature Converter** – converts between temperature units
* **Car Cost Calculator** – calculates total vehicle cost
* **Inflation/Deflation Calculator** – applies percentage changes
* **Point of Sale Calculator** – simulates a simple purchase system
* **Consortium Manager** – calculates total paid amount and remaining debt with input validation and loop control

---

## Concepts Practiced

* Variables and data types (`int`, `double`, `char`)
* Input and output (`cin`, `cout`)
* Arithmetic operations
* Conditional statements (`if`)
* Loops (`while`, `do-while`)
* Input validation
* Boolean logic (`&&`, `||`, `!=`, `==`)

---

## How to Run

1. Navigate to a project folder:

   ```
   cd project-folder
   ```

2. Compile the program:

   ```
   g++ main.cpp -o app
   ```

3. Run:

   ```
   ./app
   ```
