# Car Cost Calculator

## Description

A C++ console application that calculates the final price of a car based on its factory cost.

The program applies a 10% distributor fee and a 45% tax to determine the final consumer price.

---

## Features

* Validates input to prevent negative values
* Applies percentage-based calculations
* Allows repeated calculations using a loop
* Uses colored output for better readability

---

## Concepts Practiced

* Variables and data types (`float`)
* Input and output (`cin`, `cout`)
* Arithmetic operations
* Percentage calculations
* `while` loops (input validation)
* `do-while` loops (program repetition)

---

## How to Run

Compile and run the program using a C++ compiler.
