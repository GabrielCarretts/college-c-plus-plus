# Simple C++ Calculator

## Description

A basic C++ console calculator that performs arithmetic operations based on user input.

---

## Features

* Supports addition, subtraction, multiplication, and division
* Uses conditional statements to select operations
* Displays results with color based on value
* Allows repeated calculations

---

## Concepts Practiced

* Conditional statements (`if`, `else if`)
* Arithmetic operations
* User input handling
* Loops (`do-while`)
* Basic output formatting

---

## How to Run

Compile and run the program using a C++ compiler.
