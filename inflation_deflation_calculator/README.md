# Inflation & Deflation Calculator

## Description

A C++ console application that calculates the percentage change between two values.

It determines whether the result represents inflation, deflation, or no change.

---

## Features

* Calculates percentage variation between two values
* Classifies result as inflation, deflation, or neutral
* Formats output for readability
* Uses colors to highlight results

---

## Concepts Practiced

* Arithmetic operations
* Percentage calculations
* Conditional statements (`if`, `else if`, `else`)
* Boolean logic

---

## How to Run

Compile and run the program using a C++ compiler.
