# Array Addition

## Description
A fundamental C++ algorithm designed to practice data structures. The program takes two parallel arrays (vectors) of 5 integers inputted by the user, adds their corresponding elements together, and stores the sums in a third array. 

## Concepts Applied
* Array (Vector) declaration and memory allocation
* `for` loops for sequential data entry and output
* Parallel array processing and index management (`a[i] = x[i] + y[i]`)
* Separation of Concerns (Input, Process, Output structure)

## How to Run
1. Compile the file in your terminal:
   ```bash
   g++ array_addition.cpp -o app
   ```
2. Execute the program:
   ```bash
   ./app
   ```
