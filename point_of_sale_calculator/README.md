# Point of Sale Calculator

## Description

A C++ console application that simulates a simple point of sale system.

The program allows the user to enter multiple products, apply discounts, and calculate the final total.

---

## Features

* Processes multiple products using a loop
* Calculates discounted prices
* Keeps a running total of purchases
* Handles product names with spaces (`getline`)
* Allows repeated transactions

---

## Concepts Practiced

* Strings (`getline`)
* Input handling (`cin`, `cin.ignore`)
* Loops (`for`, `do-while`)
* Arithmetic operations
* Accumulator pattern (running total)

---

## How to Run

Compile and run the program using a C++ compiler.
